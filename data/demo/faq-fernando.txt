# FAQ - AITI Assistant (Demo para Fernando)

## Sobre a AITI Assistant

### O que é a AITI Assistant?
A AITI Assistant é um assistente virtual inteligente com tecnologia RAG (Retrieval-Augmented Generation). Funciona como um chatbot que consegue responder a perguntas específicas da sua empresa, baseando-se em documentos, políticas e FAQs que você carrega no sistema.

### Quais são as principais vantagens?
- **24/7 Disponível**: Atende clientes a qualquer hora
- **Instant Respostas**: Respostas em menos de 3 segundos
- **Preciso**: Baseado em seus próprios documentos
- **Escalável**: Suporta múltiplos idiomas e temas
- **Rastreável**: Mostra as fontes de cada resposta
- **Seguro**: Dados encriptados e privados

### Como funciona internamente?
1. Você carrega documentos (PDFs, Word, TXTs)
2. A IA transforma em "embeddings" (representações numéricas)
3. Quando alguém faz uma pergunta, o sistema procura as seções relevantes
4. A IA gera uma resposta contextualizada usando os documentos

## Implementação

### Quantos documentos posso carregar?
Sem limite. O sistema suporta desde 10 até 10.000+ documentos. Quanto mais informação, melhor a qualidade das respostas.

### Quais formatos de documento são suportados?
- PDF (mais comum)
- Word (.docx)
- Texto (.txt)
- Markdown (.md)
- CSV

### Quanto tempo demora a processar documentos?
Geralmente minutos. Um lote de 100 PDFs processa em 2-5 minutos, dependendo do tamanho.

### A IA inventa respostas quando não sabe?
Não. Se a resposta não se encontra nos documentos carregados, o sistema avisa claramente que não tem informação e oferece escalar para um humano.

## Configuração

### Preciso de API keys?
Sim. Você escolhe entre:
- **OpenAI** (GPT-4o): Mais preciso
- **Claude (Anthropic)**: Mais seguro e criativo
- Ambas têm preços similares (~$0.001 por pergunta)

### Qual é o custo mensal?
Depende do volume de perguntas, mas tipicamente:
- 100 perguntas/dia = ~€30/mês
- 1000 perguntas/dia = ~€150/mês

### Posso customizar a "personalidade" do assistente?
Sim! O "system prompt" define como ele fala. Exemplos:
- Formal para bancos
- Casual para e-commerce
- Técnico para suporte IT

## Segurança e Privacidade

### Os dados são encriptados?
Sim. Em trânsito (HTTPS) e em repouso (base de dados encriptada).

### Cumpre RGPD?
Sim. Você controla totalmente quais dados são armazenados e quem acessa.

### Posso deletar todos os dados?
Sim, com um comando. Tudo é permanentemente removido.

### Quem acede aos meus documentos?
Apenas você (e o LLM durante processamento). Nenhum acesso terceiros.

## Troubleshooting

### O assistente dá respostas imprecisas
- Verifique que os documentos estão bem escritos e claros
- Carregue mais documentos com mais contexto
- Ajuste o "confidence threshold" (confiança mínima)

### Está muito lento
- Reduza TOP_K_RESULTS (padrão: 5)
- Aumente CHUNK_SIZE (padrão: 500)
- Use um modelo LLM mais rápido (gpt-4o-mini vs gpt-4)

### Preciso re-treinar o modelo
- Não é necessário! Só carregue novos documentos
- O sistema indexa automaticamente

### Como vejo estatísticas de uso?
- Dashboard em /admin (requer login)
- API em /api/metrics
- Logs exportáveis em CSV
