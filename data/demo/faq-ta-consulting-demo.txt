## Serviços de IA

### O que é uma AITI Assistant?
É um chatbot inteligente com Retrieval-Augmented Generation (RAG). Lê seus documentos (FAQs, políticas, manuais) e responde perguntas específicas com base nesses dados.

### Como funciona o RAG?
1. O cliente fornece documentos (PDFs, Excel, Word, TXT)
2. O sistema converte para "embeddings" (vetores numéricos)
3. Quando alguém pergunta, o sistema encontra os trechos relevantes
4. A IA gera uma resposta contextualizada baseada nesses trechos
5. A fonte é sempre citada (rastreabilidade)

### Qual é a precisão?
Depende da qualidade dos documentos. Em geral:
- Documentos bem estruturados: 85-95% de precisão
- FAQs simples: 90%+
- Documentação desorganizada: 60-70%

## Automação com n8n

### O que é n8n?
É uma plataforma open-source para automação de workflows. Conecta qualquer aplicação (Gmail, Slack, CRM, ERP) sem código.

### Exemplos de automação
- Processar emails → Extrair dados → Guardar em BD
- Converter leads → Criar tickets → Notificar equipa
- Scraping de dados → Transformação → Dashboard atualizado
- Alertas automáticos → Escalação → Notificações

### Qual é o ROI?
Tipicamente 3-6 meses. Automatizar processos manuais economiza 15-20 horas/semana por funcionário.

### Posso integrar com meu ERP?
Sim! n8n conecta com: Moloni, Sage, SAP, Oracle, Microsoft Dynamics, etc.
